    Cada vez que el código hace algo y puedes obtener un valor de este, estamos ante una expresión. Pero si la porción de código no hace nada como declarar una función o una variable estamos delante de una delacaration pero si de lo contratio este te permite poder escribir lógica o realizar ciertas tareas entonces estamos delante de la precencia de un statement.

    Si no le puededes hacer un console.log(let, const, for, if) entonces es un declaration. De lo contrario si te permite hacer un console.log() es una expressión.
